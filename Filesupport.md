# File Support

- `.md`, `.markdown`: opens in the editor, with markdown preview mode available.
- `.docx`: opens as a read-only document preview in the main view.
- `.pdf`: opens as an embedded PDF preview in the main view.
- `.jpg`, `.jpeg`, `.png`, `.gif`, `.webp`, `.svg`, `.bmp`, `.ico`, `.tif`, `.tiff`, `.avif`: opens as an image preview in the main view.
- `.mp4`, `.m4v`, `.webm`, `.mov`, `.avi`, `.mkv`, `.ogv`: opens as a video player in the main view.
- `.zip`: opens like a folder, with virtual navigation into the archive contents.
- Other files: opens in the text editor as plain text.
